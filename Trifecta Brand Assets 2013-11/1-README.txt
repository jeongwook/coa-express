# Trifecta Logo Pack
## 2013 November
These logo files are for general Trifecta project use.

## Files
### Standard Bitmaps
- **Trifecta-Logo-Black-White-XL.png**
	black logo on white background
- **Trifecta-Logo-On-Black-XL.png**
	color logo on black background
- **Trifecta-Logo-Solid-XL.png**
	logo with solid colored logomark (no gradients)
- **Trifecta-Logo-Solo-XL.png**
	logo with no tagline
- **Trifecta-Logo-Standard-XL.png**
	standard version of the logo
- **Trifecta-Logo-White-Black-XL.png**
	white logo on black background

### Designer
- **TrifectaLogoStandard.eps**
	logo is a vector, suitable for print projects
- **TrifectaLogoVersions-CMYK.ai**
	layered file, print, CMYK color
- **TrifectaLogoVersions-RGB.ai**
	layered file, screen-only, RGB color
- **TrifectaLogoVersions.ai**
	layered file, print, Pantone color

## File Details
The EPS and AI files are high quality/technical *vector* versions of the logo. Use these if the program you are using supports this format. If not, use a PNG file. The PNG files have a transparent background and should be compatible with modern programs. (You can also convert PNGs to JPGs quite easily using any simple image editing program.)
